Position Setting

1) This is static position.
2) This is relative position. The element's place is reserved. The left/top/right/bottom values are w.r.t. the static position of this element. And any non-zero value indicates a shift. The default values of left and right are zero w.r.t. its static position. It doesn't change the positions of other elements around it, look at this. This shifted "this" word to bottom right.
3) This is absolute-body position, which is positioned w.r.t. the body. So all the left/top/right/bottom are relative to the entire page body. Here the element is position at top right corner of the body. The element's place is NOT reserved, which means next word starts as if this element didn't exist. The default values of left and top are NOT zero instead their defaulted to the current static position of the element if it were present.
This is absolute-parent position, which is positioned w.r.t. the immediate element in the parent hierarchy which has "position" property overridden (with a value other than "static"). Here, the element in the parent hierarchy which has "position" property overridden is yellow box. So it's position at bottom right corner of the yellow box.
